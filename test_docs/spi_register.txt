# SPI 寄存器说明(测试样本)

## SPI_CR1 寄存器(控制寄存器 1)
地址偏移: 0x00
复位值:   0x0000

位 0  CPHA - 时钟相位 (Clock Phase)
  该位选择数据采样的时钟边沿。
  0: 在第一个时钟跳变沿采样
  1: 在第二个时钟跳变沿采样

位 1  CPOL - 时钟极性 (Clock Polarity)
  该位控制空闲状态时 SCK 引脚的电平。
  0: 空闲时 SCK 为低电平
  1: 空闲时 SCK 为高电平

位 2  MSTR - 主设备选择 (Master Selection)
  0: 配置为从设备
  1: 配置为主设备

位 3  BR[2:0] - 波特率控制
  这些位配置串行时钟的波特率:
  000: fPCLK/2
  001: fPCLK/4
  010: fPCLK/8
  ...
